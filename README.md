# Data Visualization Analysis of 2025 QS World University Rankings

## 1. Project Overview
This project provides a comprehensive data visualization analysis of the 2024 QS World University Rankings. In an era of increasing global educational competition, university rankings serve as a critical benchmark for academic excellence, teaching quality, and international influence. By leveraging data visualization techniques, this analysis explores the current landscape of global higher education, helping institutions, students, and policymakers understand trends and performance across different regions and categories.

## 2. Dataset Description
Source: The dataset is sourced from Kaggle.
Scope: It covers 1,500 institutions across 104 locations.
Size: 1,498 entries and 29 variables.
